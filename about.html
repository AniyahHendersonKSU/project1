About this project

Reading the Source is a milestone-based research project for IT 3203. This first milestone establishes the site's structure and content: a homepage, two topic pages analyzing HTML and CSS in a real web template, a key-concepts glossary, a references page, and this about page. Later milestones will expand the research, refine the design, and add further analysis.

What this milestone demonstrates

  • Six pages with a consistent main menu (this tab bar) on every page
  • Proper HTML elements: headings, ordered and unordered lists, a table, and images
  • One shared CSS file (style.css) for all pages
  • Three selector types: element (p, h1), class (.card, .tab), and ID (#gutter-demo)
  • A CSS Grid layout (page structure, key-concepts cards) and a Flexbox layout (nav bar, key terms)
  • Hover effects, padding/margins, borders, color, and custom font pairing (JetBrains Mono + Inter)
