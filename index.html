Reading the Source: An Analysis of HTML5 & CSS in the W3.CSS "Cafe" Template

This site is a close reading of a real, publicly available web template — the "Cafe" template distributed with the W3.CSS framework on w3schools.com. Rather than treating a template as something to copy and fill in, this project opens it up: what HTML5 elements does it choose and why, how does its CSS actually select and style those elements, and what does that tell us about how the framework (and the developer using it) thinks about layout and design?

The goal isn't to praise or dismiss the template, but to understand it well enough to explain its decisions to someone else — including the places where it takes shortcuts, repeats itself, or relies on inline styles instead of the stylesheet doing the work.
